github怎样绑定自己的域名？

1, 在本地hexo下的source文件夹里新建一个名为CNAME的txt文本，打开，填上你的域名,保存

然后上传

hexo g

hexo d

2, 在域名提供商处，以freenom为例

新增两条记录

然后保存 Save Changes

等待数分钟，刷新浏览器，用自己的域名访问试试
